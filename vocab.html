Argument - external infromatio needed for a function to operate Block elements - Take up entire lines of screen no matter the content Condition - code in the parenthesis of a Conditional Conditional - block of code headed by an if Defining a function - using the keyword "function" to create a function Event - Anything that happens, ex. when a page loads Function call = a line of code that tells a computer you want it to perform a certain task (the function) Global variable - a variable that exists for the entire program, made without var keyword Inline elements - Only as wide as their content Local variable - a variable that only exists in the function it was created, made with var keyword Log - the name of the function Parameter - xxxx Variable - An object that represents something
