  • Argument - external information needed for a function to operate
  • Block elements - Take up entire lines of the screen no matter the content
  • Condition - code in the parenthesis of a Conditional
  • Conditional - code structures used to test if an expression is true or false
  • Defining a function - using the keyword "function" to create a function
  • Event - Anything that happens, ex. when a page loads
  • Function - a line of code that tells a computer you want it to perform a certain task (the function)
  • Global variable - a variable that exists for the entire program, made without the var keyword
  • Inline elements - Only as wide as their content
  • Local variable - a variable that only exists in the function it was created, made with the var keyword
  • Log - the name of the function
  • Parameter - placeholders for values that will be passed in a function
  • Variable - A container that stores values
